# Restaurant Review Platform

A platform where users can rate and review restaurants.

## Tech Stack

- **Java 21**
- **Spring Boot 3.5.6**
- **Spring Security** (OAuth2)
- **Elasticsearch**
- **Maven**
- **Lombok**
- **MapStruct**
